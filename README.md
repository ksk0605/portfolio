## 👋 소개
자바 & 스프링 기반의 백엔드 개발을 주력으로 하며,  
도메인 모델 설계와 서비스 최적화를 고민하는 개발자입니다.

- **기술 스택**: Java, Spring Boot, JPA, MariaDB
- **주요 기여**:
  - **ATDD/TDD 기반 개발 프로세스를 적용한 백오피스 개발**
  - **Spring RestDocs를 활용한 API 문서 자동화**  
  - **피드 조회 시 발생하는 트랜잭션 경합 문제 개선**
